# AGENT.md - سير عمل ذكاء اللعبة

## دور الـ Agent
مساعد تطوير ذكي للعبة DuckH. يقرأ SKILL.md لفهم البنية ثم ينفذ المهام مرحلةً مرحلة.

## سير العمل

### عند استلام مهمة جديدة:
```
1. اقرأ SKILL.md → فهم البنية والقواعد
2. حدد أي وحدة تتأثر (core/gfx/scene/game/ui/audio/util)
3. اقرأ الملفات ذات الصلة فقط من الوحدة
4. نفذ المهمة خطوة بخطوة مع شرح عربي
5. تأكد أن الكود يتبع قواعد SKILL.md
```

### قاعدة قراءة الملفات:
- **لا تقرأ كل الملفات** عند بدء مهمة
- اقرأ فقط ملفات الوحدة التي تعمل عليها + الترويسات التي تستخدمها
- إذا طلبت "ابنِ كاميرا" → اقرأ `gfx/Camera.h` + `gfx/VulkanContext.h` فقط
- إذا طلبت "أضف موجة جديدة" → اقرأ `game/WaveManager.h` + `game/Duck.h` فقط

## حالات اللعبة (GameState)

```
MENU ──→ PLAYING ──→ WAVE_COMPLETE ──→ PLAYING (موجة تالية)
  │          │                              │
  │          └──→ GAME_OVER ──→ MENU        │
  │                                         │
  └──→ SETTINGS ←───────────────────────────┘
```

- **MENU**: شاشة البداية، اختيار اللغة، ابدأ اللعبة
- **PLAYING**: اللعبة نشطة، البط يطير، اللاعب يصطاد
- **WAVE_COMPLETE**: بين الموجات، عرض النقاط، تجهيز الموجة التالية
- **GAME_OVER**: انتهاء اللعبة، عرض النتيجة النهائية
- **SETTINGS**: إعدادات الصوت واللغة

## نظام الموجات

| الموجة | عدد البط | السرعة | أنماط الحركة | ذخيرة إضافية |
|--------|----------|--------|-------------|-------------|
| 1      | 5        | بطيء   | خط مستقيم   | 10          |
| 2      | 8        | متوسط  | منحنى خفيف  | 8           |
| 3      | 10       | متوسط  | متعرج        | 8           |
| 4+     | +3 كل موجة | +10% | أنماط مختلطة | 6           |

## أنماط حركة البط
- `STRAIGHT` ← خط مستقيم بسرعة ثابتة
- `CURVED` ← منحنى بيضاوي
- `ZIGZAG` ← متعرج أفقي
- `DIVE` ← غطس نحو الماء ثم صعود
- `SCATTER` ← تفرّع عشوائي عند إصابة بططة قريبة

## أوامر مساعدة سريعة

| أمر | الوصف |
|-----|-------|
| "ابنِ [وحدة]" | أنشئ ملفات الوحدة مع كود أساسي |
| "وصّل [ملف]" | اربط ملفًا بالنظام الحالي |
| "اختبر البناء" | cmake --build build وتقرير النتائج |
| "أصلح الخطأ" | حل مشكلة البناء الأخيرة |
| "أضف [ميزة]" | أضف ميزة جديدة للعبة |

## ملاحظات مهمة
- **لا تنشئ ملفات بدون طلب** - نعمل معًا مرحلة مرحلة
- عند إنشاء ملف: اشرح دوره ثم اكتب الكود
- أولًا نبني البنية التحتية (core + gfx) ثم منطق اللعبة
- كل مرحلة يجب أن تبني بنجاح قبل الانتقال للتالية
