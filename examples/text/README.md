# أمثلة Text Subsystem

هذا المجلد لا يدخل في بناء اللعبة تلقائيًا. الغرض منه إعطاء أمثلة استعمال واضحة وسريعة.

الملفات:

- `TextSystemQuickStart.cpp`
  - مثال minimal يبني نصوصًا عربية ومختلطة ويطبع عدد الـ glyphs والـ vertices.
- `VulkanHudIntegrationExample.cpp`
  - مثال دمج عملي مع `vertex buffer` في HUD داخل محرك Vulkan.

## سلاسل مقترحة للتجربة

- `طلقة`
- `تلقيم 1/1`
- `طلقة 460+`
- `اضغط ENTER للفتح`
- `100% دقة`
- `1 / 3 طلقة`

## ملاحظات

- الأمثلة تستخدم `TextSystem` لأن هذا هو الـ facade الأسرع للدمج.
- إذا احتجت تخصيصًا أعمق، استخدم الطبقات الداخلية مباشرة من `src/text/`.
